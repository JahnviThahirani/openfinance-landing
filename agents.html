Capabilities

What your agent can do.

17 tools across read, write, and simulate.
01 Read 9 tools
health-check Check server and API connectivity.
get-supported-chains List every blockchain network your agent can reach.
query-markets Search markets with filters and sorting across every asset class.
get-market Fetch details for a specific market.
get-positions Get all open positions for a user.
get-position Get a single user position by ID.
get-token-balance Check token balance and approval state.
query-assets Query every tradeable asset across all supported markets.
get-price Live price for any supported asset.
02 Write 7 tools
prepare-order Prepare a market or limit order ready to sign.
prepare-deposit Prepare a deposit with all required approvals.
prepare-withdraw Prepare a withdrawal. Supports max.
prepare-borrow Prepare a borrow with a built-in health check.
prepare-repay Prepare a repay. Supports max.
prepare-supply-collateral Supply collateral to a lending market.
prepare-withdraw-collateral Withdraw collateral with a built-in health check.
03 Simulate 1 tool
simulate-transactions Dry-run transactions and inspect the resulting state before signing.
